HTML-Cmd's:

A:

- A: Align   CSS property used to set the alignment of text or block elements. Example: "text-align: [where];".

B:

- Box Model   Defines the rectangular boxes generated for elements, including margins, borders, padding, and content area. Example: div, width, height, background-color.

- Border   CSS property to create borders around an element. Example: "border: [amount]px [state / leave] [color];".

C:

- Color   Used to define text or background color. Example: "color: blue;" or "background-color: [color];".

G:

- Grundaufbau  <!DOCTYPE html> <html lang="de"> <head> <meta charset="UTF-8"> <body>

M:

- Margin   CSS property to define space outside the border of an element. Example: "margin: [amount][px / %];".

P:

- Padding   Defines the space inside an element, between its content and border. Example: "padding: [amount]px;".

R:

- Radius   Used to round the corners of an element. Example: "border-radius: [amount]px;".

S:

- Size   CSS property to define the size of an element. Example: "width: [amount]px; height: [amount]px;".

T:

- Text-Align   Used to align text within an element. Example: "text-align: [where];".

Z:

- Z-Index   Defines the stack order of elements, controlling which one appears on top. Example: "z-index: [amount];".
